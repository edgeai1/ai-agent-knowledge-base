## TL;DR

Open-source platform for AI software development agents. Provides sandboxed execution, browser, and code editing capabilities.

## Key Features

- Sandboxed code execution environment
- Built-in browser for web interaction
- Model-agnostic (works with Claude, GPT, etc.)
- Enterprise-grade open-source

## Performance (2026)

- 72% on SWE-bench Verified (with Claude 4)
- 77.6% on SWE-bench Verified (with Claude 3.5 Sonnet Thinking, own harness)

## OpenHands Index (Jan 2026)

Introduced a standardized evaluation framework for coding agents beyond just SWE-bench.

## Why It Matters

Leading open-source alternative to Devin. Named one of the top open-source AI projects of 2024. Demonstrates that open-source can compete with commercial coding agents.
