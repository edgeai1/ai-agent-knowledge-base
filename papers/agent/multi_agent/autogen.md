## TL;DR

Framework enabling multi-agent conversations with customizable agents, supporting human-in-the-loop, tool use, and code execution.

## Key Features

- Customizable conversable agents
- Flexible conversation patterns (sequential, group chat, hierarchical)
- Human-in-the-loop support
- Code execution integration

## Evolution

- AutoGen v0.4 (2025): major rewrite with event-driven architecture
- Community fork: AG2
- Microsoft's production evolution of the framework

## Why It Matters

One of the most widely adopted multi-agent frameworks. Established patterns for multi-agent conversation that influenced CrewAI, LangGraph, and others.
